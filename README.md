# api-requests-python
A basic CLI to count the results returned from an API.
```
usage: API_call.py [-h] (--URL URL | --sample) [-v]                                                                                                                                             

Call an API and display the number of returned results.

optional arguments:
  -h, --help     show this help message and exit
  --URL URL      The URL representing the API endpoint
  --sample       Call a sample API: https://petstore.swagger.io/v2/pet/findByStatus?status=available
  -v, --verbose  Display extra information about the returned data
  ```
